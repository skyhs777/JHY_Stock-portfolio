# 연금저축 포트폴리오 매일 아침 리포트

매일 아침 보유 종목(연금저축계좌)의 현재가 · 평가금액 · 비중 · 관련 뉴스를
HTML 파일로 만들고, 카카오톡 "나에게 보내기"로도 요약을 전송하는 스크립트입니다.

## 실행 환경

이 코드는 **Claude Code (사용자님 PC)**에서 실행해야 합니다.
(네이버 금융 사이트, 카카오 API 서버에 접속해야 하는데, 이 코드가 작성된
샌드박스는 그런 외부 사이트 접속이 막혀 있어서 여기서는 테스트하지 못했습니다.)

## 1단계. 패키지 설치

```powershell
pip install -r requirements.txt
```

## 2단계. 종목 정보 확인/수정

`config.py` 열어서 보유 종목(HOLDINGS)이 맞는지 확인하세요. 계좌를 CMA / IRP 로
확장하고 싶으면 같은 형식으로 추가하면 됩니다.

## 3단계. 시세/뉴스 API 동작 확인

먼저 `fetch_data.py`만 단독 실행해서 네이버 API가 정상 응답하는지 확인하세요:

```powershell
python fetch_data.py
```

- 정상이면 현재가/뉴스 딕셔너리가 출력됩니다.
- 만약 에러가 나거나 값이 이상하면, 네이버 쪽에서 API 응답 구조를 바꿨을 가능성이
  높습니다. 이 경우 Claude Code에게 "이 에러 메시지 보고 fetch_data.py 고쳐줘"라고
  요청하면 실제 응답을 보면서 바로 고칠 수 있습니다.

## 4단계. 카카오톡 연동 (최초 1회만)

1. https://developers.kakao.com 접속 → 카카오계정 로그인
2. 내 애플리케이션 → 애플리케이션 추가하기 (이름 자유, 예: 주식알림)
3. 앱 선택 → **카카오 로그인** → 활성화 ON
4. **카카오 로그인 → 동의항목** → "카카오톡 메시지 전송" 필수 동의로 설정
5. **카카오 로그인 → Redirect URI** → `https://localhost.com` 추가
   (config.py의 `KAKAO_REDIRECT_URI`와 정확히 일치해야 함)
6. 앱 요약 정보 화면에서 **REST API 키** 복사 → `config.py`의
   `KAKAO_REST_API_KEY`에 붙여넣기
7. 아래 명령 실행 → 브라우저 로그인 → 리다이렉트된 주소창의 `code=` 뒷값을
   터미널에 붙여넣기

```powershell
python kakao_auth.py
```

성공하면 `kakao_token.json` 파일이 생성됩니다. 이 파일이 있어야 카카오톡 발송이 됩니다.

## 5단계. 실행

```powershell
python main.py
```

- `portfolio_report.html`이 같은 폴더에 생성됩니다 (더블클릭하면 브라우저로 열림).
- `kakao_token.json`이 있으면 자동으로 카카오톡 "나에게 보내기"로 요약이 전송됩니다.

## 6단계. 매일 아침 자동 실행 (Windows 작업 스케줄러)

1. `Win + R` → `taskschd.msc` → Enter
2. 오른쪽 **"작업 만들기"**
3. **트리거** 탭 → 새로 만들기 → 매일, 원하는 시간(예: 오전 7시) 설정
4. **동작** 탭 → 새로 만들기 → 프로그램/스크립트에 파이썬 경로 입력
   (`where python` 명령으로 확인 가능), 인수에 `main.py` 전체 경로,
   시작 위치에 이 폴더 경로 입력
5. 확인 후 저장

이후 매일 그 시간에 자동으로 최신 시세 반영된 리포트가 만들어지고
카카오톡으로도 전송됩니다.

## 참고: HTML을 클로드(claude.ai)에서도 보고 싶다면

`portfolio_report.html` 내용을 복사해서 클로드 채팅창에 붙여넣거나 파일로
업로드하면, 클로드가 그 자리에서 렌더링해서 보여줄 수 있습니다.
